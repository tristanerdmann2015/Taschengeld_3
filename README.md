Es handelt sich um eine Taschengeld-App. Es können Aufgaben definiert werden und Preise. Dann können Zeiten eingetragen werden, wann und wie oft diese Aufgaben ausgeführt wurden. 
Das Programm berechnet dann den gesamten Verdienst des Kindes wochen- oder monatsweise. Die Abrechnung kann dann mittels Messenager Diensten (Whatsapp, E-Mail, ...) mit den Eltern geteilt werden.

programmiert in MAUI, läuft also auf Windows, Android und IOS.
